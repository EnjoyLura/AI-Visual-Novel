# 美术风格指南

版本：v0.1  
日期：2026-05-16  

## 1. 总体方向

- 风格：日系二次元
- 气质：明亮、清新、干净、柔和
- 题材关键词：校园、青春、制服、樱花、夏日、海边、河堤、花火
- 参考方向：《突然＊恋人》《苍之彼方的四重奏》

## 2. 角色美术方向

- 五位女主需要在轮廓、发色、服装气质、表情节奏和视觉记忆点上明显区分。
- 男主可以低存在感处理，但仍需有稳定的设定图，方便 CG 与宣传图使用。
- 角色整体应符合明亮清新的校园恋爱作品气质。

## 3. 背景美术方向

- 背景需要统一光照、季节、色彩饱和度和透视。
- 校园场景应清爽干净，突出春季和初夏氛围。
- 城市场景应有生活感，避免过度写实或阴暗。
- 花火大会、海边、河堤等场景应承担关键恋爱氛围。

## 4. 事件 CG 方向

- CG 优先服务关键恋爱节点、告白、路线高潮、结局回收。
- 每张 CG 都应有明确剧情作用，不只作为装饰图。
- 角色表情、姿势、光照和构图应与对应剧情情绪一致。

## 5. UI 方向

- UI 风格：清爽、轻量、校园感、色彩柔和。
- 对话框应保证长文本可读性。
- 菜单、存档、设置、鉴赏页面应接近商业 Galgame 的完整体验。
