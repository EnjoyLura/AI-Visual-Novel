# 共通线第一章：开学前后

版本：v0.6

日期：2026-05-17

本文用于将 `06_剧情结构.md` 中共通线第一章的章节大纲拆解为可进入剧本生产的场景级细纲。通用格式见 `../00_细纲模板.md`。

## 1. 章节定位

- 章节目标：建立男主“无色普通”的日常底色，依次让五位女主进入玩家视野，并把新学期从平静日常推向“关系开始变多”的状态。
- 核心情绪：清晨日常、春季新学期、旧日回忆、轻微期待、轻微不知所措。
- 章节节奏：家庭日常开场 -> 旧照片伏笔 -> 商业街过渡 -> 便利店初遇 -> 樱花道重逢 -> 家庭回收 -> 夜晚内心 -> 始业式当天 -> 校园连续事件 -> 家庭收束。
- 男主状态：不主动追求变化，但会因为细心、旧日记忆和被拜托的性格，被不同角色牵引进新关系。
- 分支状态：本章不设置关键选项，不记录候选，只做角色入口和关系基线。

## 2. 场景列表

#### C01-S01：清晨被叫醒

- 场景类型：剧情场景
- 场景功能：开场建立悠真的普通生活、小春的家庭位置，并用旧照片提前埋下千纱重逢的情绪伏笔。
- 时间地点：四月，开学前一天早晨，悠真房间。
- 登场角色：桐生悠真、千羽小春。
- 入场状态：悠真还没完全醒，小春已经习惯负责叫他起床。
- 承接关系：
  - 本章从悠真的私人空间开始，让玩家先看到他的“无色日常”和被小春照顾的生活节奏。
  - 旧照片会把本场和之后的樱花道重逢连接起来。
- 事件流程：
  1. 画面从黑场和闹钟声开始，悠真内心独白自己没有特别期待新学期。
  2. 小春敲门后进入房间，用平静语气确认悠真是否醒来。
  3. 悠真试图再赖几分钟，小春直接指出他昨晚睡得不算早。
  4. 小春拉开窗帘，春天的光和风进入房间。
  5. 悠真起身时，不小心打到书桌上的旧相框掉地上了。相框里是小时候的悠真和一个女孩在樱花道附近的合照。
  6. 悠真短暂想起小时候经常和那个女孩一起走樱花道，但没有在独白中直接说出“初恋”。
  7. 小春拿起相框，说起悠真的青梅竹马。注意到悠真在发呆，平静提醒他早餐快好了。
  8. 小春把相框轻轻放回原处，出门后又提醒发呆的悠真。悠真表示马上下去。
- 对白方向：
  - 小春称呼悠真为“哥哥”，语气平淡，不撒娇。
  - 小春的关心要包装成生活管理，例如“这样比较省事”“再不下来早餐就凉了”。
  - 悠真看到照片时要轻，不要陷入长篇回忆，只让玩家知道“这张照片对他有意义”。
- 演出提示：
  - BGM：清晨日常，轻柔。
  - SE：闹钟、敲门、窗帘声。
  - 小春立绘：家居常服，默认冷静，偶尔轻微无语。
  - 相框特写：樱花、两人合照。
- 结束状态：玩家确认小春是冷静、熟悉悠真的家庭锚点，同时知道悠真过去有一个关系很好的青梅竹马。
- 分支变量：无。

#### C01-S02：开学前一天的早餐

- 场景类型：剧情场景
- 场景功能：补足重组家庭氛围，让父母作为画面外/语音角色进入家庭日常，并自然引出“买酱油”的外出任务。
- 时间地点：同日早晨，客厅厨房。
- 登场角色：桐生悠真、千羽小春、母亲、父亲。
- 入场状态：悠真刚起床，小春已经在客厅厨房帮忙；父母准备出门上班，一家人正在吃早饭。
- 承接关系：
  - 接上 C01-S01 的“早餐快好了”，从房间自然转到客厅厨房。
  - 母亲口头拜托买酱油，下一场先接悠真出门走向商业街。
- 事件流程：
  1. 悠真来到客厅厨房，看到早餐已经摆好，小春正在帮忙分餐具。
  2. 母亲开朗地调侃悠真开学前一天还一副没睡醒的样子，顺口问他新学期有没有期待。
  3. 父亲话不多，只提醒悠真不要太勉强，想做什么可以慢慢来。
  4. 小春平静指出悠真大概率不会主动找事情做，最后还是会被别人拜托。
  5. 母亲接话说既然他今天没安排，就等会儿出门帮家里买酱油。
  6. 悠真吐槽自己刚被说“容易被拜托”，马上就被拜托了。
  7. 母亲笑着说这也是一种可靠，小春则把购物袋或零钱递给悠真。
- 对白方向：
  - 母亲开朗、八卦、会调侃，但不压迫。
  - 父亲简短随和，台词少但稳定。
  - 小春的吐槽要平静，像事实确认。
  - 悠真不要强烈拒绝，体现“被拜托就会做”的性格。
- 演出提示：
  - BG：客厅厨房，早晨光线。
  - BGM：家庭日常。
  - 小春立绘：家居常服，可搭配帮忙分餐具的手部动作。
  - 父母不出立绘，只用画面外语音或餐桌旁声音。
  - SE：餐具、热汤、椅子移动。
- 结束状态：确立家庭温和开明的基调，也给悠真外出遇见茜提供自然任务。
- 分支变量：无。

#### C01-S03：走向商业街

- 场景类型：剧情场景
- 场景功能：用移步场景承接买酱油任务，展示星见市商业街的热闹，并强化悠真在人群中的形单影只。
- 时间地点：开学前一天白天，住宅街到商业街。
- 登场角色：桐生悠真。
- 入场状态：悠真带着购物袋和零钱出门，目的明确是去商业街便利店买酱油。
- 承接关系：
  - 承接 C01-S02 的母亲拜托和小春递来的购物袋。
  - 本场把家庭空间自然移动到商业街，为 C01-S04 的便利店初遇做空间铺垫。
- 事件流程：
  1. 悠真从家门口走出，沿住宅街往商业街方向走。
  2. 路上逐渐出现成对的情侣、买菜的居民和开店准备中的商铺。
  3. 进入商业街后，街道声音变得热闹，招牌、行人和店铺构成新学期前一天的生活感。
  4. 悠真看着成群走过的情侣，短暂意识到自己总是一个人完成这些跑腿和日常小事。
  5. 他没有觉得孤独到难受，只是觉得热闹和自己之间隔着一层薄薄的距离。
  6. 便利店招牌出现在前方，悠真收回思绪，确认要先把酱油买了。
- 对白方向：
  - 以悠真短独白为主，几乎不需要对话。
  - 形单影只要轻描淡写，不写成沉重孤独。
- 演出提示：
  - BG：住宅街 -> 商业街。
  - BGM：明亮城市日常。
  - SE：无。
  - 可以用人群远景和悠真独行的构图强化空间感。
- 结束状态：悠真到达商业街便利店门口，准备进入便利店买酱油。
- 分支变量：无。

#### C01-S04：便利店买酱油

- 场景类型：剧情场景
- 场景功能：让茜第一次出场，建立她在便利店打工、谨慎、现实、不习惯被关注的第一印象。
- 时间地点：开学前一天白天，商业街便利店。
- 登场角色：桐生悠真、筿崎茜；便利店店长可用画面外声音短暂出现。
- 入场状态：悠真从商业街进入便利店，只是普通跑腿；茜正在工作状态。
- 承接关系：
  - 承接 C01-S03 的商业街移步和买酱油任务。
  - 本场结束后，悠真先回到便利店外的商业街，再决定去哪里散心。
- 事件流程：
  1. 悠真推门进入便利店，自动门声把商业街的喧闹隔在外面。
  2. 他看到同龄女生穿着便利店制服，动作熟练地补货和结账。
  3. 悠真一开始没有认出她是同校学生，只觉得她比普通高中生更习惯工作。
  4. 茜胸前的工牌被货架和制服褶皱遮住，悠真没有看清她的名字。
  5. 货架上酱油位置较高或被其他货物挡住，悠真短暂犹豫。
  6. 茜注意到后主动以店员身份询问，但语气保持距离。
  7. 悠真感谢她，茜简短回应后立刻回到工作。
  8. 店长从画面外叫茜去处理另一项工作，语气不坏但有工作压力。
  9. 悠真结账离开，注意到茜很快又投入补货和收银。
- 对白方向：
  - 茜用店员式礼貌，句子短，避免私人话题。
  - 悠真不要一上来表现过度关心，只用细心观察。
  - 茜不透露家庭压力，也不主动报上姓名，只让玩家从工作状态看出她的成熟和距离感。
- 演出提示：
  - BG：便利店。
  - BGM：轻日常，可比家庭场景稍微现实一点。
  - SE：自动门、收银机。
  - 茜立绘：便利店工作服，默认谨慎表情。
  - 便利店店长不出立绘，只用画面外工作指令或呼叫声。
- 结束状态：悠真记住“便利店里那个同龄女生”，并带着酱油离开便利店。
- 分支变量：无。

#### C01-S05：便利店外的商业街

- 场景类型：剧情场景
- 场景功能：承接便利店初遇后的余韵，让悠真从商业街自然选择散步方向，再进入樱花道相关的回忆路线。
- 时间地点：开学前一天白天，便利店外的商业街。
- 登场角色：桐生悠真。
- 入场状态：悠真买完酱油，从便利店回到商业街，手里提着购物袋。
- 承接关系：
  - 承接 C01-S04 的离店和对茜的短暂印象。
  - 本场决定“去哪散散心”，连接 C01-S06 的不知不觉走到樱花道。
- 事件流程：
  1. 悠真走出便利店，商业街的声音重新涌过来。
  2. 他低头看了看购物袋里的酱油，确认任务已经完成。
  3. 他看了看时间，觉得现在回去也太早。
  4. 悠真想到刚才那个熟练工作的同龄店员，短暂觉得自己和她的日常不太一样。
  5. 他抬头看向商业街尽头，想着要不要找个安静一点的地方散散心。
  6. 悠真没有明确目的，只是从热闹的主街拐向人少的小路。
- 对白方向：
  - 以悠真短独白为主，不需要完整对白。
  - 重点是从“热闹商业街”转向“安静旧路”的心情变化。
- 演出提示：
  - BG：便利店外商业街。
  - BGM：城市日常逐渐降低存在感。
  - SE：自动门。
- 结束状态：悠真离开商业街主路，开始没有目的地散步。
- 分支变量：无。

#### C01-S06：不知不觉走到樱花道

- 场景类型：剧情场景
- 场景功能：把“买酱油”后的普通外出转成旧日回忆入口，让樱花道重逢不再像突然切场。
- 时间地点：开学前一天白天到傍晚之间，商业街支路、住宅街、河堤附近、樱花道入口。
- 登场角色：桐生悠真。
- 入场状态：悠真提着酱油，从商业街主路拐进人少的小路，打算随便散散心。
- 承接关系：
  - 承接 C01-S05 的“找个安静一点的地方散散心”。
  - 本场把 C01-S01 的旧照片记忆带回，导向 C01-S07 的千纱重逢。
- 事件流程：
  1. 画面从商业街支路逐渐过渡到住宅街和河堤附近的小路。
  2. 悠真没有明确目的，只是顺着熟悉又陌生的路走。
  3. 周围人声逐渐减少，风声和树叶声变得明显。
  4. 看到飘落的樱花后，他才意识到自己不知不觉走到了小时候常来的樱花道。
  5. 悠真想起早上看到的旧照片，意识到这里就是照片里的地方。
  6. 他停下脚步，心里浮现“原来我还记得路”的想法。
- 对白方向：
  - 以悠真独白为主，语气轻，带一点怀旧和自嘲。
  - 不要直接说“我想起千纱”，只说“照片里的那个地方”“小时候经常来的地方”。
- 演出提示：
  - BG：商业街支路 -> 住宅街 -> 樱花道入口。
  - BGM：从轻日常逐渐转为怀旧。
  - SE：脚步、远处车声、春风、花瓣。
  - 可短暂插入旧照片画面作为记忆呼应。
- 结束状态：悠真站在樱花道上，情绪从普通跑腿转为旧日回忆，为千纱叫住他做准备。
- 分支变量：无。

#### C01-S07：樱花道重逢

- 场景类型：剧情场景
- 场景功能：让千纱出场，建立“阔别多年后的初恋青梅竹马”关系和双方距离感。
- 时间地点：开学前一天傍晚，樱花道。
- 登场角色：桐生悠真、七瀬千纱。
- 入场状态：悠真站在回忆中的樱花道上发呆；千纱先认出他。
- 承接关系：
  - 接上 C01-S06 的“走到旧照片里的地方”。
  - 结尾把悠真的动摇带回家，连接 C01-S08 小春察觉异常。
- 事件流程：
  1. 悠真看着樱花道，想起小时候有人总是走在自己前面一点的位置。
  2. 有人从身后或路旁轻声叫出“悠真”。
  3. 悠真回头，看见千纱，一时间没有立刻确认。
  4. 千纱用温柔但略紧张的语气确认他还记不记得自己。
  5. 悠真想起童年记忆和分别时的片段，意识到她是七瀬千纱。
  6. 两人简单寒暄，千纱说明自己回到星见市。
  7. 悠真在称呼上犹豫，不知道该用过去的亲近称呼，还是现在的同学式称呼。
  8. 千纱察觉他的犹豫，没有逼他，只轻轻笑着说以后会再见。
  9. 千纱离开后，悠真才意识到自己手里还提着酱油，时间也比预想晚了。
- 对白方向：
  - 千纱的温柔要带小心试探，不要完全无缝恢复青梅竹马距离。
  - 悠真的反应是惊讶、怀念、笨拙，不是冷淡。
  - 避免在第一章就讲明“初恋”，只让玩家感到过去关系不普通。
- 演出提示：
  - BG：樱花道，夕方。
  - BGM：怀旧、温柔。
  - SE：春风、花瓣。
  - 千纱立绘：春季常服，温柔微笑，略带紧张。
  - 可用短闪回，但只用模糊意象，不展开完整过去剧情。
- 结束状态：千纱重新进入悠真的生活，悠真带着明显的心绪波动回家。
- 分支变量：无。

#### C01-S08：买酱油后的客厅

- 场景类型：剧情场景
- 场景功能：用家庭场景回收买酱油任务和千纱重逢，让小春第一次察觉悠真身边出现了不属于家庭的变化。
- 时间地点：开学前一天傍晚，客厅厨房。
- 登场角色：桐生悠真、千羽小春；母亲可画面外短暂插话。
- 入场状态：悠真买完酱油回家，比普通跑腿晚了一点；小春在客厅或厨房帮忙。
- 承接关系：
  - 承接 C01-S07 的“悠真晚归且心神不宁”。
  - 本场把千纱的信息交给小春，并引出 C01-S09 夜晚内心整理。
- 事件流程：
  1. 悠真回到家，把酱油放到厨房。
  2. 小春平静确认“买个酱油用了这么久”，语气像普通事实确认。
  3. 悠真随口说天气不错，稍微绕了一下路。
  4. 小春注意到悠真的回答比平时慢，追问是不是发生了什么。
  5. 悠真犹豫后说自己在樱花道遇到了小时候认识的人。
  6. 小春短暂停顿，问“是照片里的那个人吗”或“是女孩子吗”。
  7. 悠真说是七瀬千纱，她回到星见市了。
  8. 小春没有继续追问，只提醒悠真酱油已经买回来了，晚饭可以开始准备。
  9. 母亲从画面外开朗地夸悠真跑腿辛苦，小春则看着悠真短暂沉默。
- 对白方向：
  - 小春要用“买酱油”切入，而不是直接问女生。
  - 悠真说千纱时要带一点没整理好的动摇。
  - 小春的在意要轻，表现为停顿、观察和过于平静的回答。
- 演出提示：
  - BG：客厅厨房，傍晚。
  - BGM：家庭日常中带一丝安静。
  - SE：玄关门、购物袋、酱油瓶放到台面。
  - 小春立绘：家居常服，默认冷静，短暂停顿。
  - 母亲不出立绘，只用厨房或客厅方向的画面外短暂插话。
- 结束状态：小春知道悠真重逢了过去的女孩，悠真自己也意识到这件事比普通偶遇更让他在意。

## 3. 第一章生产检查点

- 第一章分成“开学前一天”和“始业式当天”两条连续日程，避免场景像独立事件卡。
- 纯空间移动不单独编号，使用“瞬间转场”写在相邻剧情场景末尾；短独白或情绪变化仍作为剧情场景保留。
- 旧照片在 C01-S01 出现，并在 C01-S06、C01-S09、C01-S21 回收，形成千纱重逢的连续情绪线。
- 买酱油任务由母亲在 C01-S02 口头提出，经 C01-S03 商业街过渡进入便利店，再经 C01-S05 转向散步和樱花道。
- 始业式当天从家庭早餐、玄关、上学路、校门、教室、操场、体育馆、教室、办公室、学生会室连续推进，空间转场要清楚。
- 五十岚信在第一章就作为同班好友和气氛调节器登场，但不抢女主第一印象。
- 班主任通过办公室和领书任务自然引出学生会室，不让悠真无理由去见绫音。
- 茜的贫困压力不明说，只通过便利店工作强度表现。
- 茜的姓名不在第一章揭示，悠真和小春都只知道她是便利店里的同龄店员。
- 绫音的小说秘密只做物件伏笔，不在第一章揭开。
- 千夏只展示明亮和行动力，不提前表现受伤或独居孤独。
- 千纱保留“过去很重要”的气味，但不直接进入初恋确认。
- 小春每次出现都与家庭秩序、生活照顾、对悠真身边人变化的观察有关。
- 有立绘角色在画面中出现时，演出提示必须写明使用的立绘类型，如常服、校服、工作服或教师装；只用声音出现的角色需标明不出立绘。
