# 共通线第二章：角色初遇

版本：v0.7

日期：2026-05-17

本文用于将 `06_剧情结构.md` 中共通线第二章的章节大纲拆解为可进入剧本生产的场景级细纲。通用格式见 `../00_细纲模板.md`。

## 1. 章节定位

- 章节目标：让五位女主从“第一章的登场或重逢”进入悠真的日常活动范围，建立可持续出现的关系入口。
- 核心情绪：新学期日常、轻微热闹、关系增加后的不适应、各女主私人一面的初步露出。
- 章节节奏：清晨叫醒 -> 客厅早餐上学 -> 河堤上学路千纱偶遇 -> 教室旧同班日常 -> 午休绫音秘密伏笔 -> 放学后教室社团闲聊 -> 放学路小春消息 -> 夜晚房间犹豫 -> 次日早晨千纱同行 -> 教室后方信调侃 -> 次日天台茜距离感 -> 操场千夏训练 -> 第一周夜晚小春充电闲聊 -> 周末自然醒 -> 客厅早饭后去向选择 -> 周末便利店茜再会（可选）或房间漫画游戏 -> 周末傍晚一起做饭 -> 周一早晨小春叫醒 -> 周一教室熬夜闲聊与图书室委员公布 -> 下周放学后千纱同行。
- 男主状态：仍然不主动追求变化，但开始被不同地点、不同角色的日常交集包围，并意识到自己不再只是旁观者。
- 分支状态：本章不设置关键进线选项，不记录候选/解锁/确认；C02-S15 的周末去向只控制是否触发便利店小场景或留家漫画游戏；只建立第三章“学生会、田径部、图书室”三选一的前置印象，并保留茜与小春的生活侧入口。

## 2. 场景列表

#### C02-S01：清晨被小春叫醒

- 场景类型：剧情场景
- 场景功能：承接第一章夜晚收束，用小春发现悠真已经醒来却无精打采的状态开启第二章，表现她对哥哥变化的关心。
- 时间地点：新学期第二天早晨，悠真房间。
- 登场角色：桐生悠真、千羽小春。
- 入场状态：悠真已经醒了，但还坐在床边发呆；小春已经换好校服，发现哥哥迟迟没有下楼。
- 承接关系：
  - 承接 C01-S21 的夜晚兄妹对话。
  - 本场先从悠真的私人空间开始，再转入 C02-S02 的客厅早餐。
- 事件流程：
  1. 画面从清晨光线开始，闹钟已经停下，悠真坐在床边看着书桌上的课程表、开学资料和课本发呆。
  2. 小春敲门后进入房间，看到悠真已经醒着，却一副无精打采的样子。
  3. 小春没有立刻吐槽，而是先平静确认哥哥是不是身体不舒服。
  4. 悠真说自己没事，只是昨天新学期第一天信息有点多，还没完全整理好。
  5. 小春看向书桌上的开学资料和课本，判断哥哥不是困，而是在想学校里的事。
  6. 她没有追问具体是什么，只提醒悠真早餐已经好了，先吃饭再继续发呆比较有效率。
  7. 悠真被她的说法带回日常节奏，起身回应马上下楼。
- 对白方向：
  - 小春保持冷静、简洁，但第一反应是关心哥哥状态，不是直接吐槽。
  - 悠真在小春面前可以自然吐槽，表现两人的家庭熟悉感。
  - 小春不主动追问千纱或其他女生，只把关心包装成“先吃饭”“不要空着肚子想事情”。
- 演出提示：
  - BG：悠真房间，早晨。
  - BGM：家庭早晨日常，轻柔。
  - SE：敲门、床边动作、书页或纸张轻响。
  - 小春立绘：星见女子学园黑色水手服校服，默认冷静、短暂担心、恢复平静。
- 瞬间转场：悠真房间 -> 客厅厨房。
- 结束状态：悠真被小春叫醒，带着还没整理完的第一天余韵下楼吃早餐。
- 分支变量：无。

#### C02-S02：客厅早餐和上学准备

- 场景类型：剧情场景
- 场景功能：用悠真和小春两人的早餐延续家庭稳定感，并让小春通过调侃“找女朋友”替无精打采的悠真提神。
- 时间地点：新学期第二天早晨，客厅厨房。
- 登场角色：桐生悠真、千羽小春。
- 入场状态：悠真下楼吃早餐，小春已经把早晨节奏整理好，试图用日常话题让哥哥恢复精神。
- 承接关系：
  - 承接 C02-S01 的下楼吃早餐。
  - 本场从家庭空间转入上学路，连接 C02-S03 的千纱偶遇。
- 事件流程：
  1. 悠真来到客厅厨房，小春已经把简单早餐摆好，并把昨日发下来的开学资料放在他座位旁边。
  2. 悠真吃饭时仍然有点慢，小春平静观察后，问哥哥是不是新学期第二天就下决心要找女朋友了。
  3. 悠真差点被呛到，反问她为什么话题会跳到那里。
  4. 小春淡淡说，哥哥昨天提到的女生和同学数量明显增加，如果不是这样，就只能解释为哥哥终于变得受欢迎了。
  5. 悠真吐槽这两种说法都很奇怪，小春回答她只是选择一个能让哥哥清醒一点的话题。
  6. 小春顺势调侃哥哥如果真的要努力，至少先把早餐吃完，不然连上学路都走不稳。
  7. 悠真恢复吐槽节奏，说自己只是普通去上学，不是恋爱相亲活动开始。
  8. 小春没有继续追问，只在早餐结束前提醒悠真带齐学生证、便当和需要带回学校的资料。
  9. 悠真把东西收进书包，旁白带过两人在玄关换鞋后分别出门：悠真前往星见学园，小春前往星见女子学园。
- 对白方向：
  - 小春用“是不是要找女朋友”这种平静调侃帮悠真提神，但语气仍然冷静，不表现明显吃醋。
  - 悠真被话题吓到后恢复吐槽节奏，表现小春确实很懂怎样让哥哥清醒。
  - 最后提醒带齐东西即可，不再展开玄关对白。
- 演出提示：
  - BG：客厅厨房。
  - BGM：家庭早晨日常。
  - SE：餐具、书包、椅子移动。
  - 小春立绘：星见女子学园黑色水手服校服，默认冷静、轻微侧目。
- 瞬间转场：客厅厨房 -> 玄关（旁白带过换鞋分别出门）-> 住宅街（过渡）-> 河堤上学路。
- 结束状态：悠真在小春一如既往的提醒下出门，进入新学期第二天的上学路。
- 分支变量：无。

#### C02-S03：河堤上学路的青梅问候

- 场景类型：剧情场景
- 场景功能：让千纱从“樱花道重逢”和“转学生入班”转为上学路上的稳定交集，并通过称呼问题表现两人重新拉近距离的笨拙感。
- 时间地点：新学期第二天早晨，河堤上学路、校门附近。
- 登场角色：桐生悠真、七瀬千纱。
- 入场状态：悠真独自走在上学路上，仍不确定自己该如何自然面对千纱。
- 承接关系：
  - 承接 C02-S02 的上学路转场。
  - 延续 C01-S07 和 C01-S16 的千纱重逢与同班关系。
- 事件流程：
  1. 悠真走在河堤上学路上，注意到河岸、樱树和通往学校的路都和记忆中的景色重叠。
  2. 千纱从后方或河堤旁的小路自然地向悠真打招呼，称呼他为“悠真君”。
  3. 悠真一瞬间不知道该回“七瀬同学”还是“千纱”，最后还是脱口而出叫了“七瀬同学”。
  4. 千纱露出疑惑表情，温柔地问他们既然已经重逢又是同班，互相用名字称呼是不是就可以了。
  5. 悠真有些犹豫地喊出“千纱”，语气不自然得连自己都觉得别扭。
  6. 千纱觉得悠真的表现有点好笑，轻轻笑出来，但没有嘲笑他，只说这样听起来比较像以前。
  7. 两人沿河堤继续往学校走，千纱看着周围景色，说这里还是和记忆中一样。
  8. 千纱又看向悠真，补上一句“悠真君也是”，让悠真短暂不知道该怎么回应。
  9. 闲聊中，千纱说自己一直想回到这里，这句话说得很轻，却让悠真意识到她对星见市和过去的记忆并不只是怀念。
  10. 两人聊着搬家后的整理、学校和河堤风景，走着走着到了学校。
- 对白方向：
  - 千纱温柔、轻缓，主动提出互称名字，但语气不要强迫。
  - 悠真笨拙，不冷淡；他叫出“千纱”时要有明显不习惯。
  - 千纱觉得好笑是因为悠真仍和记忆中一样笨拙，不是调侃他迟钝。
  - “一直想回到这里”要轻轻带出千纱对过去和星见市的眷恋，不展开过去约定或初恋确认。
- 演出提示：
  - BG：河堤上学路 -> 校门。
  - BGM：清爽校园晨间。
  - SE：脚步、河风、远处学生谈笑。
  - 千纱立绘：校服，端正穿法，温柔微笑、疑惑、轻笑、短暂怀旧。
- 瞬间转场：校门 -> 二年级教室。
- 结束状态：千纱和悠真开始重新使用名字称呼彼此，但悠真仍对这份突然回来的亲近感有些不适应。
- 分支变量：无。

#### C02-S04：教室后方的固定闲聊圈

- 场景类型：剧情场景
- 场景功能：让千夏和五十岚信成为悠真班级日常里的固定闲聊圈，并用班主任托付的通知自然引出绫音午休场景。
- 时间地点：新学期前几天，上午课间到午休后，二年级教室后方、食堂。
- 登场角色：桐生悠真、星野千夏、五十岚信；七瀬千纱、便利店见过的女生、同班学生可作背景。
- 入场状态：新班级逐渐进入日常节奏，悠真仍习惯待在不显眼的位置。
- 承接关系：
  - 承接 C02-S03 的到校。
  - 本场把第一章的旧同班三人互动稳定为教室日常。
- 事件流程：
  1. 课间，悠真刚回到座位附近，五十岚信已经自然靠过来，把他的座位旁边当成固定闲聊点。
  2. 千夏也很自然地加入，提起早上差点又在走廊加速，被老师提醒“走廊不是热身跑道”。
  3. 信调侃千夏行动力太强，又说悠真只是站在旁边，也已经被两人默认算进聊天小组。
  4. 千夏问悠真这几天适不适应新班级，语气像对旧同班的自然关心。
  5. 悠真说班里比自己想象中热闹，千夏笑着说这样才像新学期。
  6. 远处千纱正被几个同学轻声搭话；悠真又注意到之前在便利店见过的那个女生，居然和自己也是同班。
  7. 信提到再过不久社团招新会开始，学生会、田径部、图书室最近都会很忙，到时候学校会比现在更吵。
  8. 千夏一听到社团招新就露出明显精神，顺口提到田径部也要准备招新和训练展示。
  9. 午休铃响后，信问悠真中午去哪吃午饭。
  10. 悠真说小春给自己做了便当，本来打算在教室随便解决。
  11. 信提议去食堂吃便当，那里可以买饮料和配菜。
  12. 千夏被其他同学叫去买午饭，挥手说之后告别；悠真和信一起前往食堂。
  13. 吃完午饭后，悠真和信回到教室，班主任或值日负责人把一份通知交给悠真，请他趁午休后半送到绫音所在教室。
  14. 信调侃悠真最近被各种杂事找上的概率变高，悠真吐槽自己只是刚好站在离门最近的位置。
  15. 悠真拿着通知离开教室，前往绫音所在教室。
- 对白方向：
  - 千夏明快、直接，和悠真的距离是旧同班式的轻松，不要突然亲密。
  - 信负责把班级信息、午饭去向、社团招新和悠真的被动性自然带出来。
  - 悠真负责吐槽和观察，体现他被朋友拉入班级氛围。
  - 通知任务只是午休场景入口，不作为路线倾向，不提前进入学生会、田径部或图书室。
- 演出提示：
  - BG：二年级教室后方 -> 食堂 -> 二年级教室。
  - BGM：轻快校园日常。
  - SE：课间人声、椅子移动、午休铃、食堂人声。
  - 千夏立绘：校服，运动系随性穿法，明亮笑容。
  - 五十岚信立绘：校服，熟络笑和情报通表情。
  - 千纱如远处露出，使用校服远景立绘或背景站位。
  - 便利店见过的女生如远处露出，使用校服远景立绘或背景站位，不参与正面对白；悠真此时未确认姓名。
- 结束状态：悠真意识到自己在班里已经有了固定聊天位置；社团招新作为第三章主题被轻轻预告，并由通知任务引出绫音午休场景。
- 分支变量：无。

#### C02-S05：绫音教室里的稿纸

- 场景类型：剧情场景
- 场景功能：让悠真在绫音自己的教室里看到她写稿纸的瞬间，并第一次意识到她可能在写小说。
- 时间地点：新学期前几天，午休后半，绫音所在教室、走廊。
- 登场角色：桐生悠真、白鸟绫音。
- 入场状态：午饭后，悠真带着班主任交代的通知前往隔壁班或同年级另一间教室；绫音以为午休后半教室里不会有人。
- 承接关系：
  - 承接 C02-S04 的通知任务。
  - 承接第一章 C01-S15 的台上印象和 C01-S19 的学生会室伏笔。
  - 本场让悠真初步推测绫音正在写小说，但不当面揭穿，也不让绫音承认。
- 事件流程：
  1. 午饭后，悠真拿着班主任交代的通知来到绫音所在教室，准备把通知放到讲台或指定位置。
  2. 教室里几乎没人，只有绫音一个人坐在窗边，低头在稿纸上写着什么。
  3. 悠真推门时刚好看到稿纸上的一小段内容，感觉不像学生会文件，更像带有情节和人物对白的文字。
  4. 绫音察觉有人进来后立刻用课本或文件盖住稿纸，动作比学生会工作时更明显地慌了一下。
  5. 绫音看向悠真，语气努力保持端正，却直接问他刚才没看到吧。
  6. 悠真假装没看到，只说自己是来放通知的，刚进门还没反应过来。
  7. 绫音短暂审视他的表情，随后恢复学生会长式礼貌，说通知放在讲台上就可以。
  8. 悠真把通知放好，没有追问稿纸内容，也没有表现出自己刚才已经看见一点。
  9. 绫音提醒他离开前记得关门，语气端正，但比平时更快结束对话。
  10. 悠真离开教室后，在走廊里回想刚刚看到的短句，内心判断那应该是在写小说。
  11. 悠真没想到白鸟绫音这种完美学生会长，居然还会在午休时偷偷写小说。
- 对白方向：
  - 绫音表面端正，实际有短暂动摇；她越慌越会试图恢复礼貌距离。
  - 悠真假装没看到，是出于边界感和不想让绫音难堪，不是故意欺骗制造矛盾。
  - “写小说”的判断只放在悠真走廊内心，不当面说破，也不让绫音承认。
- 演出提示：
  - BG：绫音所在教室 -> 走廊。
  - BGM：午休安静，略带疑问。
  - SE：纸笔声、门声、稿纸被盖住的轻响。
  - 绫音立绘：校服正式立绘，默认端正、短暂慌张、恢复冷静。
- 瞬间转场：走廊 -> 教室（过渡）-> 放学后的教室。
- 结束状态：悠真初步知道绫音可能在写小说，但选择替她保留这个秘密，不当面追问。
- 分支变量：无。

#### C02-S06：放学后的社团闲聊

- 场景类型：剧情场景
- 场景功能：用放学后的教室闲聊承接绫音午休事件，表现悠真仍在社团选择前犹豫。
- 时间地点：新学期前几天，放学后，二年级教室后方。
- 登场角色：桐生悠真、五十岚信、星野千夏。
- 入场状态：C02-S05 之后的放学时间，教室里学生陆续收拾书包，社团活动和放学安排的话题自然变多。
- 承接关系：
  - 承接 C02-S05 的放学后教室转场。
  - 为第三章社团招新前的“要不要加入社团”问题做轻量铺垫。
- 事件流程：
  1. 放学铃响后，教室里学生开始收拾书包，周围陆续传来“去社团”“去买东西”“直接回家”的声音。
  2. 信凑到悠真座位旁，问他今天放学有什么安排。
  3. 悠真说暂时没有，应该和平时一样直接回家。
  4. 千夏把书包背到肩上，笑着说自己要去田径部参加社团活动。
