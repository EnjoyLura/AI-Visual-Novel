# 配角设定集

版本：v0.1

日期：2026-05-17

本文用于记录非攻略角色、功能配角、家庭成员、同班同学、社团成员和其他辅助角色。主角与五位女主设定见 `04_角色设定集.md`。

## 1. 配角 A：五十岚信

- 姓名：五十岚信
- 年级：高二
- 身份：桐生悠真的同班好友，高一时起与悠真和星野千夏同班
- 登场方式：有立绘
- 核心属性：经典 Galgame 好哥们、气氛调节器、情报通、自来熟、阳光、爱凑热闹
- 角色功能：负责调节班级气氛、提供校园传闻和活动信息、推动悠真进入事件、用玩笑反衬悠真的迟钝和变化
- 关联场景：教室、走廊、社团招新、文化祭、午休闲聊
- 剧情定位：非路线角色，不承担核心情感矛盾，主要服务共通线节奏和校园生活真实感

### 外貌设定

- 发色与发型：棕色短发，发梢微微翘起，整体看起来阳光、随性，像是刚从热闹的社交场合里跑出来。
- 眼睛：明亮的棕色眼睛，眼神充满活力，像是时刻都在观察周围有没有新鲜事。
- 五官与气质：笑起来会露出整齐的牙齿，亲切感强，让人觉得他没有什么距离感。
- 身材：高大，站在人群里很容易被注意到，给人可靠但不严肃的印象。
- 校服穿法：穿校服时也偏随性，外套和领口不会整理得太端正，整体像随时能和别人搭话。
- 私服穿法：常穿灰色短袖连帽外套，内搭白色 T 恤，下身是牛仔裤，脚上穿白色运动鞋。
- 视觉关键词：棕色翘发、棕色亮眼、阳光、亲切、高大、灰色短袖连帽外套、白色 T 恤、牛仔裤、白色运动鞋。

### 说话方式

- 基本语气：轻快、熟络、话多，习惯用开玩笑的方式打开话题。
- 对外态度：自来熟，能和不同班级、不同社团的人搭上话，因此经常掌握校园里的小道消息。
- 对悠真：直接称呼“悠真”，说话不客气但没有恶意。会吐槽悠真太被动、太会被拜托，也会在关键时候自然替他打圆场。
- 情报通表现：喜欢用“我刚刚听说”“这你就不知道了吧”之类的开场把校园活动、传闻或人物关系带出来。
- 调节气氛时：遇到尴尬场面会故意夸张反应，把空气变轻；但如果对方明显认真，他也能适时收住玩笑。
- 口吻关键词：轻快、自来熟、吐槽、情报通、会看气氛、不恶意起哄。

### 角色关系

- 与桐生悠真：高一时起同班的好友。信熟悉悠真温和、被动、容易被拜托的性格，常用玩笑把他推到人群中，也会最早察觉悠真身边的关系变得热闹。
- 与星野千夏：高一时起同班。信很熟悉千夏的开朗和行动力，会自然和她搭话，也常把千夏的训练、田径部消息告诉悠真。
- 与白鸟绫音：知道她是学生会长和校园名人，对她保持一定敬畏。信可以作为悠真了解学生会评价和校园传闻的入口。
- 与七瀬千纱：作为同班同学认识千纱。信会注意到悠真和千纱之间不像普通转学生与同班同学，偶尔用玩笑试探两人的过去关系。
- 与筿崎茜：同校同学。信知道茜不太合群，也可能听说她经常打工，但不会恶意传播她的家庭情况。
- 与千羽小春：主要通过悠真产生交集。信知道悠真有一个很会照顾他的妹妹，见到小春后会被她冷静的反应压住玩笑节奏。

## 2. 配角 B：悠真的母亲

- 姓名：未定
- 身份：桐生悠真的母亲，千羽小春的继母
- 登场方式：不露脸，不描写外貌，不制作立绘；只通过画面外语音、电话、门外对话、餐桌旁的声音或简短文字信息出现
- 核心属性：开明、开朗、八卦、消息灵通、观察力强、会催儿子找女朋友
- 角色功能：负责营造和谐家庭氛围，推动家庭日常的轻喜剧节奏，同时从家长视角观察悠真和小春的关系变化
- 关联场景：客厅厨房、玄关、电话、家庭餐桌、周末家务
- 剧情定位：家庭功能配角，不制造家庭矛盾，不阻碍恋爱路线；主要用于强化悠真家庭的稳定感和小春线的微妙观察压力

### 登场与表现规则

- 不出现正脸和具体外貌描写，避免占用立绘资源。
- 可用画面外台词、电话铃声后的语音、厨房方向传来的声音或手机消息表现存在感。
- 出场时以生活化小事切入，例如让悠真买东西、提醒吃饭、问今天学校怎么样。
- 在小春线中可以通过语气和停顿表现她已经察觉兄妹之间的变化，但不直接点破。
- 不作为反对者处理。即使察觉悠真和小春的关系变得微妙，也优先保持开明和观察态度。

### 说话方式

- 基本语气：开朗、轻快、亲近，带一点家长式调侃。
- 对悠真：会催他更主动一点，常用半开玩笑的方式问有没有喜欢的女生、学校里有没有可爱的同学。
- 对小春：自然亲近，尊重小春的懂事和冷静，也会担心她太会照顾别人而忽略自己。
- 八卦表现：消息灵通，能从邻居、学校通知、家长群或日常对话中知道不少事情，但不会恶意传播。
- 观察表现：会注意悠真听到某个女生名字时的反应，也会注意小春是否比平时沉默。
- 口吻关键词：开朗、调侃、关心、八卦、敏锐、开明。

### 角色关系

- 与桐生悠真：亲生母子。她理解悠真温和被动的性格，会用轻松调侃鼓励他主动接触别人，也会在他迷茫时用开明态度支持他。
- 与千羽小春：继母与继女。两人关系自然和谐，母亲把小春当作家庭成员看待，但也能察觉小春过度懂事和压抑的一面。
- 与千羽父亲：再婚夫妻。两人共同维持稳定普通的家庭氛围，不在故事中制造重组家庭冲突。
- 与五位女主：主要通过悠真的讲述或家庭对话间接了解。她会对悠真身边突然变多的女生表现出好奇和调侃。

## 3. 配角 C：千羽父亲

- 姓名：未定
- 身份：千羽小春的亲生父亲，桐生悠真的继父
- 登场方式：不露脸，不描写外貌，不制作立绘；只通过画面外语音、电话、玄关处的简短对话或家庭餐桌旁的声音出现
- 核心属性：开明、随和、话不多、稳重、重视家庭稳定
- 角色功能：作为家庭里的安定支点，用少量台词鼓励悠真，也让重组家庭保持温和可信的背景
- 关联场景：客厅厨房、玄关、电话、家庭餐桌、父母外出或加班前后的声音
- 剧情定位：家庭功能配角，不制造冲突，不承担强剧情转折；主要通过简短话语表达对悠真的信任和鼓励

### 登场与表现规则

- 不出现正脸和具体外貌描写，避免占用立绘资源。
- 出场频率低于悠真的母亲，台词更少，但每次出现应给人稳定可靠的感觉。
- 可以通过“出门前一句话”“晚饭时短暂回应”“电话里简单确认”来表现家庭存在感。
- 小春线中不主动质问两人的关系，保持父亲式的沉默观察和宽厚态度。
- 不让他说教过多。父亲的作用是简短鼓励，而不是替悠真做决定。

### 说话方式

- 基本语气：平稳、随和、简短，不主动展开长篇谈话。
- 对悠真：把悠真视为家人，愿意鼓励他去尝试、去选择，也尊重他的节奏。
- 对小春：作为亲生父亲，关心小春但不过度干涉，知道她懂事，也希望她不要把所有事都自己压住。
- 鼓励方式：不讲大道理，常用“你自己想清楚就好”“需要的话可以说”“别太勉强”这类短句表达支持。
- 口吻关键词：随和、少言、稳重、鼓励、尊重、不压迫。

### 角色关系

- 与桐生悠真：继父与继子。关系稳定和谐，不强调血缘差异；他把悠真当作家庭成员，并用简短话语支持悠真的选择。
- 与千羽小春：亲生父女。父亲理解小春的懂事，也知道她不太会主动表达需求，因此会在关键时候用简短话语关心她。
- 与悠真的母亲：再婚夫妻。两人共同维持家庭日常的稳定，家庭氛围普通、开明、没有明显对立。
- 与五位女主：基本不直接接触，主要作为家庭背景存在。
